## Descrição

Project-gun é um jogo desenvolvido com a javascript, 
tendo como base o uso da tag canvas que permite fazer animações por uma api 
de javascript, permitindo manipular esse eelemnto atraves de um contexto, os 
scripts são separados por responsabilidade no codigo, assim cada uma tendo uma função.
